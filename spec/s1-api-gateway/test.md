## 二、验收标准

### 2.1 优先级定义

| 级别 | 说明 | 通过标准 |
|------|------|----------|
| P0 | 关键功能 | 100% 通过 |
| P1 | 重要功能 | >= 95% 通过 |
| P2 | 次要功能 | >= 90% 通过 |

### 2.2 缺陷等级

| 等级 | 说明 | 处理要求 |
|------|------|----------|
| Critical | 阻塞性缺陷 | 必须修复 |
| Major | 严重缺陷 | 必须修复 |
| Minor | 轻微缺陷 | 建议修复 |
| Trivial | 提示性缺陷 | 可选修复 |

### 2.3 准出标准

- P0 用例通过率：100%
- P1 用例通过率：>= 95%
- P2 用例通过率：>= 90%
- Critical 缺陷：0 个
- Major 缺陷：<= 5 个
